##### Advantages
- Fault Tolerance
- No over-clocked server
##### Load Balancer Algos
- Round Robin
	- Distribute the load evenly
	- Low variance in response time
	- Uniform infrastructure
- Weighted Round Robin
	- Distribute the load according to the capacity of the API servers
	- Low variance in response time
- Least Active Connections
	- High variance in response time
	- Analytics workload
